# Formatting & Linting

## Formatting

- Helps with code readability and consistency
- Uses `black` to format the code
- Uses `isort` to sort the imports

```bash
make format
```

## Linting

- Helps with code readability and consistency
- Uses `flake8` to lint the code

```bash
make lint
```

## Type Checking

- Helps with code readability and consistency
- Uses `mypy` to type check the code

```bash
make mypy
```
